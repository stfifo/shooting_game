# Strikers 1945+ 스타일 종스크롤 슈팅 게임

Strikers 1945+ 스타일의 종스크롤 슈팅 게임. 빌드 도구 없이 브라우저에서 바로 실행된다.

## 실행 방법

```
game.html 파일을 브라우저에서 직접 열면 즉시 실행
```

별도 서버, 빌드, 패키지 설치 없음. 수정 후 F5로 새로고침.

## 조작법

| 키 | 동작 |
|---|---|
| 화살표 / WASD | 이동 |
| 자동 | 연사 |
| Z | 폭탄 (전체 적 제거 / 보스 25% 데미지) |
| X | 스킬 (킬 20회 충전 후 파이어볼 발사) |
| Space | 일시정지 |

## 게임 구조

총 **30웨이브**, 5웨이브마다 보스 등장 (총 6회).

| 웨이브 | 배경 | 보스 |
|---|---|---|
| 1 – 4 | 지구 | — |
| **5** | 대기권 전환 | 보스 tier 1 (HP 170) |
| 6 – 9 | 대기권 | — |
| **10** | 화성 전환 | 보스 tier 2 (HP 260) |
| 11 – 14 | 화성 | — |
| **15** | 천왕성·해왕성 전환 | 보스 tier 3 (HP 350) |
| 16 – 19 | 천왕성·해왕성 | — |
| **20** | 명왕성·카이퍼벨트 전환 | 보스 tier 4 (HP 440) |
| 21 – 24 | 명왕성·카이퍼벨트 | — |
| **25** | 은하 전환 | 보스 tier 5 (HP 530, 혼돈) |
| 26 – 29 | 은하 | — |
| **30** | 외계 행성 | 보스 tier 6 (HP 620, 최종·혼돈) |

## 무기 시스템

피격 시 SINGLE로 복귀. 아이템 P로 업그레이드.

```
SINGLE → DOUBLE → TRIPLE → SPREAD → FIREBALL
```

## 아이템

| 아이템 | 효과 |
|---|---|
| P | 무기 업그레이드 |
| B | 폭탄 +1 |
| F | 윙맨 (15초) |
| L | 목숨 +1 |
| R | 속사 8초 (연사속도 ×2.2) |
| N | 관통탄 10초 (보라색) |
| S | 쉴드 12초 (피격 1회 흡수) |
